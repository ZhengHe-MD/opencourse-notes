Quick notes

  • 上节课用的 pinhole
    • 前后都贴了胶布。3D 打印材料有点问题
    • 在中间钻孔，越小越好
  • 上节课拍的图片
    • 有哪些问题？
      • 墙上有些斜斜的线条 (还有彩虹)
      • white balance
      • blurry
      • 灰尘 (很大)
      • 曝光问题
        • 右上角有过曝
        • 右下角有暗角 (黑斑)
    • 老师实际上做了一点后期：拉高了对比度。因为原始图片偏黑白
    • why blurry
      • 曝光时间长 (23s) 人可能移动
      • Pinhole 本身的质量差
  • file organization
    • os, file system, directory hierarchy
    • Software, apple aperture
    • combination
      • 用文件系统管理原始文件，用 organization software 索引
    • Keywords (like hash tag)
      • easy to explode, become hard to maintain
  • raw images
    • 同一个品牌相机，可能都会有不同的 raw 格式
    • 不同的 raw 格式需要单独解析。
    • Raw 格式并不是一个容易移植 (portable) 的格式
    • 在很久以后，也许你就找不到打开以前 raw 格式的工具。不便于你翻阅照片
    • adobe dng
  • metadata
    • Exif
    • GPS: pros & cons, flickr (do a whole lot of analysis on the metadata, to prevent leaking data)
      • Warning/good to know
    • Photoshop/other software:
      • will add extra metadata (manipulated by xxx, version xxx)
      • can decide whether to keep the metadata when saving an image
  • Software
    • take photos in intervals
    • remote shutter button
  • Ethics of editing photos
    • Mis-represent the scene
    • photo tampering history
    • Projects will specify where the line is
  • lights
    • duality of waves and particles
      • Waves: color
      • Particles: intensity
      • visible spectrum
        • coincide with the maximum electronic-meganetic output from the sun
      • Polarization: direction of waves
  • the anatomy of eyes
    • Fovea: consentrated cells
      • 越中心，cones 越多。甚至在最中心，没有 rods，因此在夜晚，想捕捉更多的细节，不要聚焦在亮点上，而是稍微移开一些视线
    • rods
      • 只有一种，无法区分色彩。因此如果在很暗的地方，色彩区分很不明显。
    • Cones
    • wave lengths
      • more cones on M (green) and L (red), less on S (blue)
      • rods don't react to red signals。在暗房开红光，你的视觉还能感知到细节；如果是白光，你的 cones 会立即被干掉。导致无法看到任何细节
